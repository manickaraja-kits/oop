Java Control Statements

If-Else Statement

The if-else statement executes different blocks of code based on conditions.

public class IfElseExample { public static void main(String[] args) { int age = 18; if (age >= 18) { System.out.println("You are eligible to vote!"); } else if (age >= 16) { System.out.println("You can get a driver's license!"); } else { System.out.println("You are too young for both!"); } } }
Click "Check Eligibility" to see the result

Nested If Statement

You can nest if statements inside other if statements for complex conditions.

public class NestedIfExample { public static void main(String[] args) { int score = 85; char grade; if (score >= 90) { grade = 'A'; } else if (score >= 80) { if (score >= 85) { grade = 'B'; System.out.println("Good job!"); } else { grade = 'B'; } } else if (score >= 70) { grade = 'C'; } else { grade = 'F'; } System.out.println("Grade: " + grade); } }

For Loop

The for loop is used when you know the number of iterations in advance.

public class ForLoopExample { public static void main(String[] args) { System.out.println("Counting from 1 to 5:"); for (int i = 1; i <= 5; i++) { System.out.println("Count: " + i); } System.out.println("Loop finished!"); } }
Click "Run For Loop" to see the result

While Loop

The while loop continues as long as the condition is true.

public class WhileLoopExample { public static void main(String[] args) { int i = 1; int sum = 0; while (i <= 10) { sum += i; System.out.println("Adding " + i + ", sum = " + sum); i++; } System.out.println("Final sum: " + sum); } }
Click "Run While Loop" to see the result

Do-While Loop

The do-while loop executes at least once, then checks the condition.

public class DoWhileExample { public static void main(String[] args) { int number = 5; do { System.out.println("Number: " + number); number--; } while (number > 0); System.out.println("Countdown finished!"); } }
Click "Run Do-While Loop" to see the result

Break Statement

The break statement is used to exit a loop or switch statement.

public class BreakExample { public static void main(String[] args) { for (int i = 1; i <= 10; i++) { if (i == 5) { System.out.println("Breaking at i = " + i); break; } System.out.println("i = " + i); } System.out.println("Loop ended"); } }
Click "Run Break Example" to see the result

Continue Statement

The continue statement skips the current iteration and moves to the next one.

public class ContinueExample { public static void main(String[] args) { for (int i = 1; i <= 10; i++) { if (i % 2 == 0) { continue; // Skip even numbers } System.out.println("Odd number: " + i); } } }
Click "Run Continue Example" to see the result

Switch Statement

The switch statement selects one of many code blocks to execute.

public class SwitchExample { public static void main(String[] args) { int dayOfWeek = 3; String dayName; switch (dayOfWeek) { case 1: dayName = "Monday"; break; case 2: dayName = "Tuesday"; break; case 3: dayName = "Wednesday"; break; case 4: dayName = "Thursday"; break; case 5: dayName = "Friday"; break; case 6: dayName = "Saturday"; break; case 7: dayName = "Sunday"; break; default: dayName = "Invalid day"; } System.out.println("Day: " + dayName); } }
Click "Run Switch Example" to see the result

Enhanced Switch (Java 12+)

Modern Java supports enhanced switch expressions with arrow syntax.

public class EnhancedSwitchExample { public static void main(String[] args) { int month = 3; String season = switch (month) { case 12, 1, 2 -> "Winter"; case 3, 4, 5 -> "Spring"; case 6, 7, 8 -> "Summer"; case 9, 10, 11 -> "Fall"; default -> "Invalid month"; }; System.out.println("Season: " + season); } }
